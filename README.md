# MangaFox-Manga-Downloader-NodeJS
MangaFox Manga Downloader Using NodeJS

###first open a terminal in the folder and enter the commands
####$ npm install
####$ node app -n <Manga Name> -c <Chapter Serial Number>

###Example

####$ node app - n "bleach" -c 502
